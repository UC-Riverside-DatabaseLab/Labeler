About The Website

CrowdTagger is a website developed by two students, Algan Binggi Rustinyato and Sara AlaeeJordehi, who worked as intern and PhD student, respectively, under supervision of Dr. Vagelis Hristidis at University of California Riverside and Dr. Sean Young at University of California Los Angeles.
The site was built to help with the collaborative labeling and its goal is to provide a platform for labelers from around the world to share the labeling tasks. The users of the site are assigned tasks with posts and labels. Each task has a description and labelers have to choose labels for different posts of different tasks. Their work will be evaluated by an algorithm. The overall score of all labelings for a single task based on the posts in that task will be reported in the site.

Roles and Activities

There are two roles assigned to people in this site : Admin and Labeler. Labelers choose labels for each post in the tasks assigned to them. Admins monitor the labelers' progress and observe the statistics.
Admin's activities include :
  • Create and Edit Tasks
  • Create and Edit Quizzes for the Tasks
  • Add Labelers to the Tasks
  • Monitor the Labelers' Progress
    • Labeler's activities include :
  • Taking Quiz
  • Labeling Posts

    • Terms Explanation

    There are two terms used in this site : Task and quiz. Task is created by admin and finished by labelers, quiz is created along with task and must be finished by each labeler before he/she can start answering the task.
    A typical Task has:
  • Title: must be unique.
  • Description (optional)
  • Labelers: a group of user that are assigned to this task, each labeler will be assigned some posts and need to respond with the most appropriate label.
  • Labels: a list of possible labels that each labeler need to choose from.
  • Posts: a list of posts that each labeler need to respond to.
  • Evaluation: evaluation measured from the results of all labelers, can be viewed by the creator.
    • A typical Quiz has:
  • Title: must be unique.
  • Description (optional)
  • Passmark: a mark the labeler must achieve in order to answer the task.
  • Post and Label: taking a quiz is similar to taking a task, but in the quiz each question has onlt one correct answer, labeler must answer the quiz and get enough credit in order to take the task.

    • How to Use the Website

    If you are an admin :
    1. Create your quiz here.(optional)
    2. Prepare all your data in a csv file or create a database connection here.
    3. Create your task here, you can use the quiz you created in the first step and the data you prepared in the second step.
    4. Once the task is successfully created, you can monitor the progress of the task here, wait until at least one labeler finish his/her task.
    5. Then you can view the evaluation result here, you can also download the evaluation results in a csv file.
    If you are a labeler :
    1. Take your quiz here if your task has one.
    2. Once you complete the quiz, you can continue to take the task. The progress bar will turn green once you finish the minimal requirement of the task and you can click exit to quit current task(you don't need to take all the questions for one task), but you can always come back and take more questions.

    How to create new task

    To create new task, you need to use this page. Then perform following steps:
    1. Enter the title of the task, e.g., "my-task".
    2. Enter the task description(optional).
    3. Enter min number of posts, your posts should contains at least this number of rows.
    4. Enter min number of labelers, you should at least assign task to this number of labelers.
    5. Load post and label data.
      • You can choose to load post/label from a csv file(in this sample format)
      • Or load from existing database(see next section).
      • You can view your data imediately after you upload your csv file or choose your post table.
    6. Choose the quiz for this task, or create one here(optional).
    7. Enter the labelers to participate in the task, separated by line.
    8. Click submit once you think everything is correct.
    If you see this page after submission, that means your task is successfully created!

    How to manage database connection

    To create new connection to database, you need to use this page or use the popup window in the create task page. Then perform following steps:
    1. Select the type of database you want to connect to(currently support mysql and asterixdb).
    2. Enter all neccessary information to connect to the database.
    3. Click submit once you think everything is correct.
    If you see this page after submission, that means your have successfully connect to the database, otherwise please carefully check the information you provided and make sure that your database allows connection from the host server(169.235.27.196).
    To manage existing connection to database, you need to use this page or use the popup window in the create task page. Then perform following steps:
    1. Click the connection name if you want to see the details of the connection.
    2. You can view a snippet(max 10 rows) of each table in current database.
    3. Click on the trash icon if you want to delete the connection.
    To create task from database connections, you need to perform following steps:
    1. Choose load from database in the create task page and make sure you have already create at least one database connection through above steps.
    2. Select database name, post table name, post column name sequentially. Your first choice will limit your selection of the second choice(cascade selection), so if you want to start over, simply click the small crossmark to delete current choice.
    3. Type in your labels, separated by ";".
    You can also peek your selected table once you select your post tabel name, if you meet any display error like the choice won't show, please refresh.The implementation of popup window is not robust enough and may have several bugs, so considering use direct link for secured operation.

    How to edit existing task

    To edit existing task, you need to use this page. Then perform following steps:
    1. Choose a task in the task list and click on it's title.
    2. Then you can change the title, description, quiz or order of current task, changing the min posts/labeler is not recommended.
    3. Click submit once you think everything is correct.
    To edit labelers for existing task, you need to use this page. Then perform following steps:
    1. Choose a task in the task list and click on edit.
    2. Then you can change the existing labelers by typing new new labeler or remove existing one.
    3. Click submit once you think everything is correct.
    You should see a similar success page if your edit is successful.

    How to create/edit quiz

    To create new quiz, you need to use this page. Then perform following steps:
    1. Enter the title of the quiz, e.g., "my-quiz".
    2. Enter the quiz description(optional).
    3. Enter min number of questions, your questions should contains at least this number of rows.
    4. Enter the pass mark.
    5. Load question data from a csv file(in this sample format)
    6. Click submit once you think everything is correct.
    To edit current quiz, you need to use this page. Then perform following steps:
    1. Choose a quiz in the quiz list and click on edit.
    2. Then you can change the title, description, passmark of current quiz, changing the min number of questions is not recommended.
    3. Click submit once you think everything is correct.
    You should see a similar success page if your edit is successful.

    How to evaluate the task

    To evaluate current task, you need to use this page. Then perform following steps:
    1. Click task name or see evaluation to see details of current task.
    2. The first table displays Kappa Evaluation of all the labelers that finish the task. You can view the pairwise kappa cofficient and total Krippendorff's alpha or download the result as a csv file in the name of kappa-${tasktitle}-${taskid}.csv. see example here.
    3. The second table displays detailed reponse of all the labelers that finish the task. You can view the response of each labeler and the majority vote or download the result as a csv file in the name of eval-${tasktitle}-${taskid}.csv. see example here.

    How to take the task as a labeler

    To take the task, you need to create a task and assign it to some accounts.
    • If you log in as a non-admin account, you will be automatically redirect to take task page.
    • If you log in as a admin account, you need to either click the labeler view button on the right top of your page, or click here.
    • Taking the task is straightfoward, simply click on the task you want to take, and you can start work on the multiple choice questions.
